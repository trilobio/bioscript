Module fragment

The fragment module optimally fragments DNA for GoldenGate systems.

Optimal fragmentation is accomplished by using empirical fidelity data derived by NEB in the paper "Comprehensive Profiling of Four Base Overhang Ligation Fidelity by T4 DNA Ligase and Application to DNA Assembly". We use the data associated with T4 ligase @ 37c for 1hr. Paper link: https://doi.org/10.1021/acssynbio.8b00333

next_overhangs (overhangs)
fragment.next_overhangs and fragment.next_overhang have some fundamental limitations. The algorithm to find the most effective next overhang is rather naive, since it doesn't actually look more than 1 step ahead. For example, if AAAA and ATGC are of equal efficiency this round, but ATGC eliminates a lot of possible overhangs next round, fragment.next_overhangs cannot differentiate between AAAA and ATGC
