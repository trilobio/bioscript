Module fasta

This module provides a parser for working with FASTA formatted genetic sequences.

FASTA is a flat text file format developed in 1985 to store nucleotide and amino acid sequences. It is extremely simple and well-supported across many languages.

Functions

parse (input) Parses a given FASTA string into a list of FASTA tables.

parse (input)
Parses a given FASTA string into a list of FASTA tables.

Parameters:

  • input the input FASTA string

Returns:

    a list of FASTA tables, each containing an identifier and a sequence.

Usage:

    local fastas = fasta.parse(">test\natgtaa")
